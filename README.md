# 🚀 Riccardo Rodio - Personal Website & Product Catalog

Benvenuti nel repository del mio sito personale e catalogo prodotti. Il progetto è una Single Page Application (SPA) moderna, performante e reattiva, progettata per la gestione e la visualizzazione di un inventario prodotti con funzionalità avanzate di calcolo prezzi e monitoraggio, oltre che a tutte le informazioni utili per scoprire se i prodotti e l'attività Herbalife fanno per te.

🌐 **Live Demo:** [https://www.riccardorodio.com](https://www.riccardorodio.com)

---

## ✨ Caratteristiche Principali

- **Catalogo Dinamico:** Visualizzazione prodotti con sistema di ricerca e filtri.
- **Calcolo Prezzi Intelligente:** Algoritmo personalizzato per il calcolo dei prezzi in base al ruolo utente (Cliente, Distributore, ecc.) e livelli di marketing.
- **Interfaccia Zoom:** Componente custom per l'ingrandimento delle immagini dei prodotti tramite React Portals e Framer Motion.
- **Analytics & GTM:** Integrazione completa con Google Tag Manager per il tracciamento degli eventi (aggiunta al carrello, cambio quantità, zoom immagini).
- **Design Responsive:** Ottimizzato per desktop, tablet e mobile grazie a Tailwind CSS.
- **Dark Mode:** Supporto nativo per il tema scuro.

---

## 🛠️ Tech Stack

Il progetto è costruito con le tecnologie web più moderne:

- **Framework:** [React](https://reactjs.org/) (Vite)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animazioni:** [Framer Motion](https://www.framer.com/motion/)
- **Icone:** [Lucide React](https://lucide.dev/)
- **Deployment:** [Vercel](https://vercel.com/)
- **Tracciamento:** Google Tag Manager (GTM)

## 🚀 Installazione Locale

Se desideri clonare il progetto e farlo girare sul tuo computer, segui questi passaggi:

1. **Clona il repository:**

```
git clone https://github.com/Rickyrodio93/riccardorodio.com.git
```

2. **Entra nella cartella:**

```
cd riccardorodio.com
```

3. **Installa le dipendenze:**

```
npm install
```

4. **Avvia il server di sviluppo:**

```
npm run dev
```

Il sito sarà visibile su **localhost:5173.**

---

## 📁 Struttura del Progetto

- src/components/: Contiene tutti i componenti React (Navbar Riga, Tabella, ecc.).

- src/utils/: Funzioni di utilità, configurazione GTM e logica di calcolo.

- public/: Asset statici come icone e immagini dei prodotti.

- vercel.json: Configurazione per il routing e i redirect su Vercel.
